困った時は「困った時の分岐表」を見てください。 それでも分からなければ管理者（オーナー）に連絡してください。

🗓 毎日やること

毎朝（開店前）

  1. 管理画面を開く
  2. ホーム画面 を表示
  3. 「今日の予約件数」「要対応」「次の予約」を見る
  4. 要対応が 0以上 だったら、顧客カルテタブで内容を確認する

毎晩（閉店前）

  1. 予約管理タブ で「今日」を選ぶ
  2. 「✅ 来店済み」を押し忘れている予約がないかチェック
  3. 無断キャンセルがあれば「⚠️ 無断」で記録
  4. 「要対応」が残っていれば、対応してから「対応済み」に変更

🏠 ホーム画面で見る場所

ホームには3つのカードが並びます。

① 📋 今日やること

  • 左の数字：今日の予約の合計件数
  • 真ん中（赤）：要対応のお客様の人数。0より大きければ優先
  • 「次の予約：○○ 様」：これから来るお客様の時刻と名前

② ⚡ よく使う入口

  • 「📋 予約管理」：今日の予約一覧へ
  • 「🗂 顧客カルテ」：お客様の情報・履歴を見る
  • 「🌐 予約ページを開く」：お客様が見ている予約画面を別タブで確認

③ 📈 売上ハイライト（任意）

- 今月の予約数・来店数の小さな数字が出ます。閉店前のチェックに使えます。

📅 今日の予約を見る

開き方

  • ホームの 「📋 予約管理」 ボタン、または上の 「予約・顧客」 タブ

期間タブ

  • 📅 今日 … 今日の予約だけ
  • 今週 … 今週分
  • 今後すべて … 未来の予約全部

普段は 「今日」 タブで作業します。 「あの予約どこ？」となったら 「今後すべて」 に切り替え。

予約カードの見方

場所 意味
左上の時刻 予約の開始〜終了時間
右上のバッジ 🆕新規 / 🔁○回目（リピート回数）
「📋問診済み」「⏳問診 未記入」 問診の状況
「✅来店済み」「⚠️無断キャンセル」 来店結果

絞り込み

  • アーティスト：担当者ごとに絞り込み
  • 問診ステータス：「問診：未記入」を選ぶと 問診リマインドを送るべき人だけ が出る

⚠️ 要対応のお客様に返信する

「要対応」とは

お客様がLINEで「担当者に聞きたい」「不安なんです」と送ってきたとき、AIが自動で「スタッフが確認します」と答えて、このリストに名前が上がります。

このとき AI の自動返信は止まっています。 スタッフが直接 LINE で返信するか、電話で対応してください。

一覧の開き方

  • ホームの 「要対応を見る」 ボタン、または 顧客カルテ タブ → 「要対応」フィルターをタップ

お客様カルテ内の確認

  • カルテの上のほうに 「⚠️ 有人対応中：○○（理由）」 が出ます
  • 来店回数、最終来店日、累計利用額もここで見られます
  • 過去のメモ・対応履歴も時系列で表示されます

LINEへの返信

LINE公式アカウントから普通にスタッフが返信してください（この管理画面からLINE送信はしません）。

✅ 対応完了にする（押すと AI 返信が再開）

⚠️ このボタンが一番大切です。 対応が終わったら 必ずその日のうちに「対応済み」 にしてください。 押さないと、AIがそのお客様にずっと返信しない状態が続きます。

手順

  1. 顧客カルテ タブで対象のお客様をタップ
  2. 画面の下のほうにスクロール
  3. 「✏️ メモ・区分・有人対応の更新」 パネルを探す
  4. 「有人対応」 のドロップダウンを 「対応済み」 に変える
  5. 「有人対応の理由」欄に一言メモを残す（例：電話で説明済み、来店時に確認）
  6. 「この内容で更新する」 を押す

押したあとに何が起きるか

  • カルテの「要対応」バッジが消える
  • AI がそのお客様に自動返信できる状態に戻る ✅
  • ホームの「要対応」カウントが −1 される

注意

  • 「対応中（要フォロー）」のまま残すと ずっと AI が止まったまま です
  • 必ずその日のうちに「対応済み」へ

🗂 お客様カルテを見る

入り方

  • 顧客カルテ タブ → 検索ボックスにお名前 or LINE名 → 名前をタップ

画面に出る内容

  • 上：お名前・来店回数・累計利用額・最終来店日
  • 📝 同意書（電子署名）：署名済みなら日付＋写真使用可否
  • 📋 事前問診票：アレルギーなど問診内容
  • 背景メモ：スタッフが残した対応メモの累積
  • 来店・対応履歴：時系列のタイムライン

メモを追記する

  1. 下部 「背景メモに追記」 に一言書く
  2. 「この内容で更新する」 を押す
  3. 自動で日付が付いて、既存メモは消えずに追加される

📨 問診を送る

いつ送る？

予約カードに 「⏳問診 未記入」 が出ているお客様で、来店までまだ時間がある とき。

手順

  1. 予約カード内の 「📨 問診リマインド」 ボタンを押す
  2. 「このお客様に問診リマインドをLINEで送信しますか？」が出る
  3. 「OK」 を押す
  4. お客様のLINEに問診フォームのリンクが届く

送れない時

- 「LINEと連携していないため送れませんでした」 → 電話予約のお客様は送れません。来店時に紙の問診を渡してください。

📝 同意書を送る

いつ送る？

初来店 のお客様で、来店の 数日前 に送るとスムーズ（署名はスマホで完結）。

手順

  1. 予約カード内の 「📝 同意書を送る」 ボタンを押す
  2. 「施術同意書の記入リンクをLINEで送信しますか？」が出る
  3. 「OK」 を押す
  4. お客様のLINEに電子同意書リンクが届く
  5. お客様が署名すると、カルテに 「✅ 署名済み」 が表示される

送れない時

- 電話予約のお客様はLINE連携がないため送れません。紙の同意書 を来店時に使ってください。

✅ 来店を記録する

来店済み

  1. 予約カード内の 「✅ 来店済み」 ボタンを押す
  2. 確認画面で 「OK」
  3. カードに「✅来店済み」バッジ
  4. カルテの 来店回数・累計利用額が自動で更新 される

無断キャンセル

  1. 「⚠️ 無断」 ボタンを押す
  2. 「OK」
  3. カードに「⚠️無断キャンセル」バッジ

料金について

来店済みを押すと メニューの料金が累計に加算 されます。 特別な金額調整が必要な場合は、オーナー・管理者に伝えてください。

🔄 予約変更・キャンセル対応

「日時を変えたい」と連絡がきた

  1. お客様から希望の新しい日時を聞く
  2. 現在の予約カードの 「❌ キャンセル」 → 「OK」
  3. 改めて新しい日時で予約を作る： - 「📞 ＋ 電話予約を登録する」 ボタン - お客様名・電話番号・担当・メニュー・日時を入力 → 「登録する」

「キャンセルしたい」と連絡がきた

  1. 予約カードの 「❌ キャンセル」 ボタン
  2. 確認画面：「この予約をキャンセルしますか？カレンダーの枠も解放します。（お客様への自動連絡はしません）」
  3. 「OK」
  4. お客様への自動連絡はシステムから送られない ので、LINEか電話で一言伝えてください

電話で予約を受けた

  1. 「📞 ＋ 電話予約を登録する」 ボタンを押す
  2. お客様名・フリガナ・電話番号を入力
  3. 担当アーティスト を選ぶ（先に選ばないとメニューが出ません）
  4. メニュー・新規/リタッチ・日時を選ぶ
  5. 「登録する」

🆘 困った時の分岐表

状況 まず確認する 対処
今日の予約が見つからない 期間タブが「今日」か 「今後すべて」に切替
問診リマインドが送れない アラートの文言 電話予約のお客様は送れない。紙で対応
同意書が送れない アラートの文言 電話予約のお客様は送れない。紙で対応
「権限がありません」 自分の権限 管理者（オーナー）に連絡
画面が真っ白／ボタンが灰色 ログイン状態 ページを再読み込み（⌘+R）。それでも×ならオーナー連絡
要対応の数が減らない カルテの「有人対応」が「対応済み」になったか カルテで再度「対応済み」→「この内容で更新する」
お客様名が検索で出ない 検索ワード（LINE名／お名前） 別の名前で試す。電話予約はLINE名が無い場合あり
来店ボタンを間違えて押した 取り消し方法 オーナー・管理者に依頼（スタッフでは取り消せない）

🚫 絶対にやってはいけないこと

スタッフ画面に最初から「触れる導線がない」もの

安心してください。スタッフ権限では そもそも画面に出ません。

  • LINE接続（鍵の登録） … 設定タブの「LINE接続」パネルは表示されません
  • スタッフの追加・削除・パスワード発行 … 「ユーザー管理」タブは管理者のみ
  • メニュー・料金の変更 … 「設定を変更」ボタンが灰色で押せません
  • アーティストの情報・担当メニューの変更 … 同上
  • 初期設定の本番開始ボタン … 表示されません

スタッフが気をつけること（誤操作防止）

⚠️ 注意
❌ キャンセル ボタン 「OK」を押したら取り消せない。日時と名前を声に出して確認してから押す
⚠️ 無断 ボタン 同様に取り消し不可。本当に無断か確認してから押す
「この内容で更新する」 有人対応が「対応不要」になっていないか確認（誤ってAI返信を再開しないように）
キャンセル後 お客様へ自動連絡されない。LINE or 電話で必ず一言 伝える
---

困った時の最後の連絡先：オーナー（システム管理者）まで。画面のスクショを撮って一緒に送ると早く解決します。
