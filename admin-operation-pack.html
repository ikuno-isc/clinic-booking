💉 メニュー変更

一覧を開く

お店の設定 → メニューとアーティスト → 💉 メニュー・料金

編集する

  1. 変えたいメニューの 「✏️ 設定を変更」 を押す
  2. 編集できる項目：
    • メニュー名
    • 通常料金 / モニター料金
    • 所要時間（新規 / リタッチ）
    • 表示順
    • 有効 / 無効
  3. 「💾 保存」 を押す → 「✅ 保存しました」が出れば完了

追加する

  1. 一覧下部の 「＋ 新規メニューを追加」 を開く
  2. 名前・料金・所要時間を入力
  3. 「＋ 追加」 を押す
  4. 追加後は アーティスト タブで担当を割り当てる（やらないと予約ページに出てきません）

非表示にする・削除する

  • 一時的に外す → 編集して 「有効」のチェックを外す → 「💾 保存」
  • 完全に消す → 「🗑 削除」（予約や担当割り当てで使用中は削除できません）

👩‍⚕️ アーティスト変更

一覧を開く

お店の設定 → メニューとアーティスト → 👩‍⚕️ アーティスト

編集する

  1. 対象アーティストの 「✏️ 設定を変更」
  2. 編集項目：
    • 表示名（「漢字（ローマ字）」推奨。例：「大下（maho）」）
    • 自己紹介
    • 指名料
    • 公開 / 非公開
  3. 「💾 保存」

担当できるメニューと施術時間を設定する

編集欄の下のほうの 「担当できるメニューと施術時間」 で：

  1. このアーティストが施術できるメニューにチェック
  2. 新規 / リタッチの所要時間（分）を入力。空欄なら60分
  3. 「💾 担当メニューを保存」

💡 担当メニューが1件も紐付いていないアーティストは、予約ページに出てきません。

追加する

  1. 一覧下部の 「＋ 新規アーティストを追加」
  2. 名前を入力 → 「＋ 追加」
  3. 追加後、写真と担当メニューを設定する（次項）

プロフィール写真を変更する

  1. 編集欄を開く
  2. 「📷 写真を変更」 → 端末の画像選択が開く
  3. 自動で縮小されてアップロードされる
  4. 「✅ 写真を更新しました」 → 予約ページに即反映
  5. 消したいときは 「🗑 写真を消す」

停止する（予約ページから外す）

編集欄の 「停止する」 を押す → 予約ページから消える。再開も同じボタン。

🗓 シフト変更（カレンダー化済み）

シフト編集を開く

お店の設定 → メニューとアーティスト → 🗓 シフト → 対象アーティストの 「🗓 シフトを編集」

① 基本シフト（曜日パターン）

画面上部の表で曜日ごとに設定：

  • 列：曜日 / 休 / 区間1 / 区間2
  • 「休」 にチェック → その曜日はまるごと予約不可
  • 区間は 「09:00〜12:00 / 14:00〜18:00」 のように 昼休みを空ける 使い方ができる
  • 何も入れない曜日 → 受付時間内ずっと予約可
  • 「💾 基本シフトを保存」

② 月カレンダーで特定日の例外を設定する

基本シフトの下にカレンダー：

  • ◀ 前月 / 今月 / 翌月 ▶ で月を切り替え
  • セルの色：
    • 白＝曜日パターン通り
    • 薄黄（「例外」バッジ）＝例外日が登録済み
    • 薄赤＝休み
  • 日付セルをタップ → ポップアップが開く
  • 「休み」にチェック → その日だけ休み
  • または時刻を入力 → その日だけ特別な時間
  • 「💾 この日を保存」

③ 曜日パターンに戻す

日付ポップアップ内の 「🔁 曜日パターンに戻す」 → その日の例外を消して基本シフトに戻る

④ 未保存のまま画面を離れようとした場合

「未保存のシフトがあります。破棄しますか？」 の確認が出る。

🖼 症例写真の追加・公開承認（管理者のみ）

追加する

  1. お店の設定 → アーティスト → 対象の 「✏️ 設定を変更」
  2. 編集欄下部の 「症例写真」 セクション
  3. 「📷 症例写真を追加」 → 画像を選択（JPEG/PNG/WebP・5MBまで）
  4. アップロード完了後は 「未承認」 状態で一覧に出ます

承認して公開する

  • 未承認写真の 「✅ 公開する」 を押す → 予約ページの症例ギャラリーに即反映
  • 元に戻すには 「🙈 非公開にする」

削除する

「🗑 削除」 → ストレージからも削除されます（戻せません）

⚠️ 「アップロードしただけでは非公開」。必ず「公開する」を押すまでお客様には見えません。

🏪 店舗情報変更

店舗カードを開く

お店の設定 → 🏪 お店の基本

編集する

  1. 店舗カードの 「✏️ 設定を変更」
  2. 編集項目：店舗名（予約ページタイトルにも反映）
  3. 「💾 保存」

注意

  • 店舗名を変えると 予約ページ・症例写真ページのタイトル にも反映される
  • テナント複製で作った店もここから名前変更可能

⏰ 営業時間変更

編集する

お店の設定 → 🏪 お店の基本 → 店舗カードの 「✏️ 設定を変更」

編集できる項目

  • 予約受付の開始時刻（HH:MM）
  • 最終受付（この時刻の枠まで予約可）
  • 予約の刻み（5〜120分。例：60分刻み＝毎正時に枠が出る）
  • 席数（同時に施術できる人数）
  • 定休日（曜日 or 日付）

保存後の確認

  • 必ず予約ページを開いて空き枠が想定どおりか確認（「お店の設定 → システム情報」の予約ページリンクからジャンプ）

👥 スタッフ追加・権限管理

開く

お店の設定 → 👥 スタッフ（オーナー / 管理者のみ表示）

追加する

  1. 「＋ スタッフを追加（ログイン発行）」 フォームに：
    • メールアドレス
    • 表示名
    • 役割（オーナー / 管理者 / スタッフ）
    • 担当店舗
    • 暗証番号（パスワード。「🎲 自動発行」 ボタンで安全な文字列を作れます）
  2. 「＋ 追加してログインを発行」
  3. 新規アカウントの場合、暗証番号が 画面に1回だけ 表示されます。本人に安全に伝えてください（LINEや口頭など）
  4. 既に別店舗で使っているメールアドレスの場合は、確認後にこの店舗へ所属追加できます。この場合、本人の暗証番号は変わりません
  5. 暗証番号はどこにも保存されないため再表示できません

権限の種類

役割できること
owner（オーナー）全操作。最上位
admin（管理者）スタッフ追加・設定変更可
staff（スタッフ）予約・カルテ中心。お店の設定は一部閲覧のみ

役割・店舗・有効/無効を変更する

  • 一覧の各行で直接 「保存」 ボタン
  • 「無効にする」 ボタン → そのスタッフはログイン不可になります

暗証番号を再発行する

  • 一覧の 「🔑 暗証番号を再発行」 から、新しい暗証番号を設定できます
  • 再発行後は、本人に新しい暗証番号を必ず伝えてください
  • 複数店舗に所属しているスタッフの暗証番号は全店舗共通です。複数店舗所属者の再発行は、全体管理者（platform_admin）のみ可能です

📊 集客分析（集客リンク / 友達追加 / 日別表示）

開く

上部タブの 📣 集客

流入元リンクを発行する

  1. 「➕ 流入元を追加」 で表示名（例：「遠山Instagram」「久留米チラシ」）を入力
  2. 「＋ 追加してリンク発行」
  3. LINEで開く集客リンク が発行される
  4. このリンクを SNSプロフィール・チラシ に貼ると、どこからLINE追加・予約に来たか自動で記録されます
  5. 店舗のLINE予約ページ（LIFF）が未設定の場合、集客リンクは発行できません。先に お店の設定 → 初期設定 でLIFFを登録してください

流入分析の見方

下部の 「📊 流入分析」：

表示モード

  • 📊 合計表示：流入元ごとの合計を見る
  • 📅 日別表示：1日ごとに、友達追加・申込・キャンセル・来店・売上を見る

合計表示の基準切替

  • 📥 申込日ベース（既定）：その月に 予約が入った 件数 → 集客の効き目を見る用
  • 📅 施術日ベース：その月に 施術された 件数 → 来店・売上の実績を見る用
  • 💚 友達追加日ベース：その月に LINE友達追加された 人数 → InstagramなどからLINEに来た人数を見る用

💡 SNSキャンペーンを6月にやったら、まず 「友達追加日ベース」 でLINE追加数を見て、その後 「申込日ベース」 で予約につながったかを見ます。

🆘 トラブル対応

「要対応」のお客様への対応

  1. 顧客カルテ → 「要対応」 フィルタ
  2. お客様を開く
  3. メモを書く → 有人対応を 「対応済み」 に → 「この内容で更新する」
  4. AI自動返信が再開

予約の二重登録が疑われる

  1. 予約管理 タブで同じ日時・担当のカードがないか
  2. 重複していたら片方を 「キャンセル」
  3. お客様にLINEで連絡

予約のキャンセル処理

  • 予約カードの 「キャンセル」 → Googleカレンダーからも自動削除（gcal-sync）
  • LINE通知も自動でお客様に送信されます

取得失敗・画面エラー

  • まずページをリロード（⌘+R）
  • 改善しない → 画面のスクショを撮ってシステム管理者に連絡

ログインできない

  • パスワードを忘れた → システム管理者に連絡（パスワードリセット）
  • 何度も失敗 → システム管理者に連絡

🏢 マルチテナント（複数店舗の管理・システム管理者専用）

店の切り替え

  • ログイン後ヘッダーに現在の店名
  • 複数店舗を管理できる権限（platform_admin）の場合、店選びコンソール が最初に出る
  • 操作したい店を選んでから各機能を使う

全体管理者（platform_admin）だけの操作

  • LINE接続（鍵の登録）
  • テナント複製・テナント削除
  • 🧭 初期設定 内の LINE接続パネル

✅ 月次最終確認チェック欄

（毎月1日に実施推奨。印刷して使うときは日付を書いてチェック）

✓確認項目
☐メニューの料金・所要時間に誤りがない
☐アーティストの担当メニュー紐付けが正しい
☐シフト（基本＋例外日）が翌月分まで登録されている
☐Googleカレンダーに過去の削除漏れがない
☐AI活用レポートで「要対応の未完了」が0件

☐テストデータ（テストメニュー・テスト予約）が残っていない

困った時の最後の連絡先：システム管理者まで。画面のスクショと一緒に送ると早く解決します。
